A digital image may contain objects that can be made up of multiple regions concerning different material properties, physical or chemical attributes, thus, segmented simplicial meshes are generated to represent the partitioned regions. Typical methods need a pre-processed image before the mesh generation, we propose an algorithm that generates the segmented simplicial quality mesh directly from a 2D or 3D image using almost the same approach, also we present its GPU-version.
